# 🕷️ 一键爬虫 - 自动分类版

## 三个版本

| 版本 | 文件 | 功能 | 适用 |
|------|------|------|------|
| 小白版 | `start_easy.bat` | 双击运行，自动分类 | 新手入门 |
| 专业版 | `start_pro.bat` | 自动分类 + Web界面 | 日常使用 |
| 完整版 | `start.bat` | 所有功能 | 高级用户 |

## 推荐使用：专业版

### 功能

- ✅ 自动分类（视频/音频/图片/文档）
- ✅ Web实时监控界面
- ✅ 多线程快速爬取
- ✅ 自动生成CSV/JSON/SQLite
- ✅ 双击运行

### 使用方法

1. 双击 `start_pro.bat`
2. 输入网址
3. 选择模式
4. 浏览器自动打开监控界面

### Web监控界面

访问 http://localhost:8088

功能：
- 📊 实时统计（已爬取/失败/链接数）
- 📁 分类统计（视频/音频/图片/文档）
- 📈 进度条
- 📋 最近爬取的页面列表
- 🔄 自动刷新（每5秒）

## 输出结构

```
output/
├── 视频/          # 自动分类的视频文件
│   └── 列表.csv   # 可用Excel打开
├── 音频/          # 自动分类的音频文件
│   └── 列表.csv
├── 图片/          # 自动分类的图片文件
│   └── 列表.csv
├── 文档/          # 自动分类的文档文件
│   └── 列表.csv
├── 压缩包/        # 自动分类的压缩包
│   └── 列表.csv
├── 网页/          # 爬取的网页（HTML+文本）
├── 爬取结果.json  # 完整数据（JSON格式）
└── 爬取结果.db    # 完整数据（SQLite格式）
```

## 模式说明

| 模式 | 页面数 | 深度 | 适用场景 |
|------|--------|------|----------|
| 快速模式 | 50页 | 2层 | 快速测试 |
| 标准模式 | 200页 | 3层 | 日常使用 |
| 完整模式 | 500页 | 4层 | 深度爬取 |

## 自动分类规则

| 类型 | 识别的文件格式 |
|------|---------------|
| 视频 | mp4, avi, mov, wmv, flv, mkv, webm |
| 音频 | mp3, wav, ogg, flac, aac, wma |
| 图片 | jpg, jpeg, png, gif, bmp, svg, webp |
| 文档 | pdf, doc, docx, xls, xlsx, ppt, pptx, txt |
| 压缩包 | zip, rar, 7z, tar, gz, bz2 |

## 查看结果

### 查看CSV列表
- 每个分类目录下都有 `列表.csv`
- 可以用Excel或WPS打开

### 查看SQLite数据库
- 下载 [DB Browser for SQLite](https://sqlitebrowser.org/)
- 打开 `爬取结果.db`

### 查看网页
- `网页/` 目录下有HTML和文本两种格式
- HTML可以直接用浏览器打开

## 注意事项

1. 请遵守网站的robots.txt规则
2. 控制爬取速度，避免给网站造成压力
3. 部分网站可能有反爬机制
4. 建议先用快速模式测试

## 常见问题

**Q: 为什么有些链接没有爬取？**
A: 可能是域名不在允许范围内，或者文件类型被过滤了。

**Q: 如何修改爬取设置？**
A: 编辑对应的 .py 文件顶部的配置区。

**Q: 输出文件在哪里？**
A: 在当前目录下的 `output` 文件夹。

**Q: 如何停止爬取？**
A: 按 `Ctrl+C` 或直接关闭窗口。

**Q: Web界面打不开？**
A: 确保安装了flask：`pip install flask`
